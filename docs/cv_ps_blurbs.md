# CV / PS wording

## CV title
**Reanalysis of a Within-Subject Flanker Experiment**

## CV bullets
- Reorganized raw spreadsheet exports from a 2 x 2 within-subject Eriksen flanker experiment into block-level and subject-level behavioral datasets.
- Reproduced repeated-measures ANOVA results and extended the analysis with a block-level mixed-effects model and descriptive error checks.
- Documented data provenance, validation, and small-sample limitations in a reproducible GitHub workflow.

## PS sentence
Reworking a classroom flanker experiment into a reproducible analysis project made me more attentive to the relationship between data structure and inference: what initially looked like a standard repeated-measures ANOVA became a question about reconstruction of raw exports, hierarchical organization, robustness, and the limits of small-sample behavioral claims.
