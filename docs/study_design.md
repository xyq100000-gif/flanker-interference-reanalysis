# Study Design

- Task: Eriksen flanker task
- Design: 2 x 2 within-subject factorial design
- Participants: 12
- Factors:
  - `arrow_load`: 5 arrows (`low`) vs 7 arrows (`high`)
  - `flanker_consistency`: internally consistent vs internally inconsistent non-target arrows
- Outcome:
  - `congruency_effect_ms = rt_incongruent_ms - rt_congruent_ms`

The main inferential target is whether congruency effects differ across the two manipulated factors.

This repository is methods-oriented. It focuses on data restructuring, reproducible analysis, and careful discussion of small-sample limitations.
