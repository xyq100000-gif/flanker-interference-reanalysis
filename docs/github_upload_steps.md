# GitHub Upload Steps

1. Create a new public repository on GitHub named `flanker-interference-reanalysis`.
2. Do **not** upload the original course PDF, the assignment brief, or the classroom photos.
3. Upload the contents of this folder instead.
4. Keep the attribution statement in `docs/attribution.md`.
5. Set the repository description to:

   `Reanalysis of a classroom Eriksen flanker experiment: raw-to-tidy behavioral data, repeated-measures ANOVA, and block-level mixed-effects modeling.`

6. Pin the repository only if you do not already have stronger research projects.
7. In the GitHub About section, add topics such as:
   - `behavioral-data`
   - `reaction-time`
   - `repeated-measures`
   - `mixed-effects-models`
   - `reproducible-research`
   - `python`

8. For the first commit message, use:

   `Initial public reanalysis: raw-to-tidy pipeline, validation, ANOVA, mixed model, and figures`

9. After upload, check the README on GitHub and confirm that:
   - the validation statement is visible near the top
   - the limits section is visible
   - the project is framed as a methods-oriented reanalysis

10. In your CV, list the project under `Selected Projects` or `Research-Oriented Coursework`, not under `Publications`.
